# План: Timeline View для расписания

## Что нужно сделать

### 1. Новый дизайн расписания
Вместо таблицы с часовыми слотами сделать визуальную шкалу:
- Время слева (15:00, 15:30, 16:00...)
- Бронирования отображаются как блоки на шкале
- Точное позиционирование по времени начала
- Высота блока = длительность бронирования

### 2. Пример как на скриншоте:
```
15:00 ├─[15:00-15:30] Бронь 1
15:30 ├─
16:00 ├─[16:15-17:15] Бронь 2  ← начало в 16:15, блок смещён вниз
16:30 ├─
17:00 ├─[17:30-18:30] Бронь 3  ← начало в 17:30
```

### 3. Техническая реализация
- CSS Grid или Absolute positioning
- Каждый час = 60px или 100px высоты
- Позиция блока = (часы * 60) + минуты
- Высота блока = длительность в минутах

### 4. Файлы для изменения
- `components/schedule-table.tsx` - полная переделка
- Возможно использовать библиотеку @fullcalendar/timegrid

## Варианты реализации

### Вариант A: FullCalendar (быстрее)
Использовать FullCalendar с плагином timegrid:
- Уже есть интеграция FullCalendar в проекте
- Добавить vertical resource view
- Бронирования отображаются автоматически с точным временем

### Вариант B: Кастомная компонента (гибче)
Своя реализация на CSS Grid:
- Полный контроль над дизайном
- Можно сделать как на скриншоте
- Легче кастомизировать

## Рекомендация
Вариант B - кастомная компонента, чтобы получить точно такой же дизайн как на скриншоте.
